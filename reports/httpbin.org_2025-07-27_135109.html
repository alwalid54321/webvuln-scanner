High Severity Findings (7)

Missing Content-Security-Policy Header

URL https://httpbin.org
Plugin security_headers
Severity HIGH
CWECWE-1021

Description

The Content-Security-Policy header is missing. Helps prevent Cross-Site Scripting (XSS) and data injection attacks.

Evidence

No Content-Security-Policy header was found in the response

Remediation

Implement a strong Content-Security-Policy that restricts sources of executable scripts, objects, frames, etc.. Suggested value: default-src 'self'; script-src 'self'; object-src 'none'; frame-ancestors 'self'; form-action 'self'; base-uri 'self'; img-src 'self' data:; connect-src 'self'; Reference: https://developer.mozilla.org/en-US/docs/Web/HTTP/CSP

Missing Strict-Transport-Security Header

URL https://httpbin.org
Plugin security_headers
Severity MEDIUM
CWECWE-319

Description

Evidence

Remediation
